# Professional Login Form Project

This repository contains a simple, professionally styled login form built using HTML and CSS.

## Project Structure
- index.html: Main structure and content.
- style.css: Styling for the form, ensuring a clean look with borders and modern aesthetics.
- vercel.json: Configuration file for deployment on Vercel.

## Deployment Instructions (Vercel)
This project is configured for deployment on Vercel. To deploy:
1. Ensure you have the Vercel CLI installed or use the GitHub integration.
2. Run `vercel` in the root directory.
